### Project Overview
본 프로젝트는 간단한 게시판을 구현한 프로젝트입니다. 기본적인 CRUD에 관련된 RestAPI를 BackEnd에 구현하였고, Fornt 단에서 Axios라이브러리를 활용한 Http통신으로 서버와 데이터를 주고 받도록 하였습니다.
